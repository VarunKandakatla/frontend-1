Types of Selectors:

  1. Simple Selectors
  2. Combinator selectors
  3. Pseudo-class Selectors
  4. Pseudo-element Selectors
  5. Attribute Selectors

1.Simple Selector

  1. Tag :- Selects all elements within the tags
  2. id :- id is Unique always. It selects the elements with the given id (#id-name)
  3. class :- selects the elements which has same class names. (.class_name)
  4. Universal(*) :- It selects all the elements (*)
  5. Grouping :- Diffenrent combinations can be grouped by adding ,

2.Combinator Selector

combinations of all simple selectors

  1. General Sibling Selector (~) :- Targets all next sibling tags of a specific tag.
  2. Adjacent Sibling Selector (+) :- Targets immediate next sibling of a tag.
  3. Child Selector (>) :- Targets next children of a specific tag.
  4. descendant Space (space) :- Targets all children and grand children of a specific tag.

3.Pseudo-class selectors

Which gives temporary effects

  1. :link :- for unvisited links
  2. :visited :- for visited links
  3. :active :- for links
  4. :focus :- only works with input
  5. :hover :- changes the color when moving cursor
  6. :nth-child() :- selects all childs in some manner
  7. :first-child() :- changes the first child of every attribute
  8. :last-child()

    9. and many more....

4.Pseudo element selectors

This elements not used widely.It is used to style specific element

  1. ::before :- content:url()
  2. ::after :- it is used for content
  3. ::first-line :- targets all first lines of particular element
  4. ::first-letter :- targets all first lettr of element
  5. ::marker :- styles bullets in list

5.Attribute Selector

it is used in input tags like type,target

  1. Syntax: selector[attribute="value"]{ }
